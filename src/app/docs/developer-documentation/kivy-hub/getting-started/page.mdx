# Getting Started

Welcome! This guide will help you set up and run the project locally on your machine.

## Prerequisites

Before you begin, make sure you have the following installed:

- **[Node.js](https://nodejs.org/)** (v18 or later recommended)
- **[npm](https://www.npmjs.com/)**, **[yarn](https://yarnpkg.com/)** or **[bun](https://bun.com)**
- (Optional) **[Git](https://git-scm.com/)** if cloning from a repository

## Installation

1. **Clone the repository:**

```bash
git clone https://github.com/Kivy-Infoeducatie/Kivy-Hub.git
cd Kivy-Hub
```

2. **Install dependencies**

```bash
# Using npm
npm install

# using yarn
yarn install

# Using bun
bun install
```

3. **Run development server**

```bash
# Using npm
npm run dev

# using yarn
yarn dev

# Using bun
bun dev
```
