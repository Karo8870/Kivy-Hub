# Water Intake

Consumed water can be managed within the Kivy App by accessing the Water Intake and Water Logging pages.

## Water Intake

Through the Water Intake page, we can see metrics about the water consumed today. We can see a 30-day history as well as water logged in the past few days

![cook-ai](/docs/mobile/water-intake.jpeg)

## Log Water

As mentioned, we can log water consumed using either the precision wheel which allows custom values or use one of the predefined buttons.

![cook-ai](/docs/mobile/log-water.jpeg)
